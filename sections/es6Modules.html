ES6 Modules

ES6 Modules ermöglichen wiederverwendbare Code-Dateien mit Named Exports und Imports.

tools.js - Exports

// Export: Diese Funktionen sind in anderen Dateien nutzbar
export const add = (a, b) => a + b;
export const multiply = (a, b) => a * b;

main.js - Imports

// Import: Spezifische Funktionen aus tools.js laden
import { add, multiply } from './tools.js';

console.log(add(5, 3));        // 8
console.log(multiply(4, 7));   // 28

index.html

<!-- type="module" aktiviert ES6 Modul-Syntax -->
<script type="module" src="main.js"></script>

Default Export

// tools.js: Ein Hauptexport pro Datei
export default function calculate(a, b) {
    return a + b;
}

// main.js: Import ohne Klammern, Name frei wählbar
import calc from './tools.js';
console.log(calc(10, 20));     // 30

Named vs. Default Export: Named Exports erlauben mehrere Exports pro Datei, Default Export ist der Hauptexport (nur einer pro Datei).
