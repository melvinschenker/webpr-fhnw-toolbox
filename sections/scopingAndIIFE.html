Scoping und IIFE

Eine IIFE (Immediately Invoked Function Expression) ist eine Funktion, die sofort nach ihrer Definition ausgeführt wird. Sie ermöglicht es, private Variablen zu erstellen, die von außen nicht zugänglich sind (Encapsulation).

Beispiel: Counter mit privatem Zustand

const Counter = (() => {
  let n = 0;  // private Variable
  return {
    inc: () => ++n,
    dec: () => --n,
    value: () => n,
  };
})();

// Zugriff auf die Methoden:
console.log(Counter.value()); // 0 (Startwert)
Counter.inc();                // erhöht auf 1
Counter.inc();                // erhöht auf 2
console.log(Counter.value()); // 2
Counter.dec();                // verringert auf 1
console.log(Counter.value()); // 1

Erklärung: Die Variable n ist innerhalb der IIFE gekapselt und kann nur über die zurückgegebenen Methoden inc(), dec() und value() manipuliert bzw. abgefragt werden. Ein direkter Zugriff auf n von außen ist nicht möglich.
